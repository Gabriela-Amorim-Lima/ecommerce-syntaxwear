## Sobre o projeto

O **SyntaxWear** é um e-commerce fictício de tênis e sneakers, criado para fins de estudo, portfólio e demonstração de boas práticas em HTML e CSS moderno. O site é totalmente estático, responsivo e fácil de personalizar.

---

## ✨ Diferenciais

- Layout moderno e responsivo
- Componentização de estilos CSS
- Uso de variáveis globais para fácil customização
- Ícones em SVG e imagens otimizadas
- Navegação simples e intuitiva

## 🚀 Como rodar o projeto

1. **Clone o repositório:**
    ```bash
    git clone <url-do-repositorio>
    ```
2. **Abra o arquivo `index.html`** no seu navegador (basta dar duplo clique ou arrastar para o navegador).

> Não é necessário backend ou instalação de dependências. Todo o site é estático.

## 🎨 Tecnologias e recursos

- **HTML5**
- **CSS3** (componentização e variáveis)
- **Google Fonts** ([Ubuntu](https://fonts.google.com/specimen/Ubuntu), [Saira Stencil](https://fonts.google.com/specimen/Saira+Stencil))
- Imagens ilustrativas e ícones em SVG

---

## 🗂️ Principais arquivos

- `index.html`: Estrutura principal do site
- `css/reset.css`: Reset de estilos para navegadores
- `css/variables.css`: Variáveis globais de fonte e cores
- `css/base.css`: Estilos base e utilitários
- `css/components/`: Estilos por componente (header, footer, hero, categorias, grid de produtos)
- `images/`: Imagens de produtos, banners, ícones e logotipo

---

## ⚙️ Personalização

- **Imagens:** Substitua os arquivos em `images/` para trocar banners, produtos ou ícones.
- **Cores e fontes:** Edite `css/variables.css` para mudar o visual global.
- **Novos produtos/categorias:** Edite o HTML em `index.html` e, se necessário, crie novos estilos em `css/components/`.
